# Color Festival
----------
- [Codechef](https://www.codechef.com/problems/BOP2)
## Code 
```
#include <bits/stdc++.h>
using namespace std;
int main() {
    int t;
    cin >> t;
    while (t--) {
        int n;
        cin >> n;
        vector <int> a(n);
        for (int i = 0; i < n; i++) cin >> a[i];
        unordered_set<int> s(a.begin(), a.end());
        cout << s.size() << endl;
    }
}
```
