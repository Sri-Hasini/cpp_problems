# C. Replacement
- [Codeforces](https://codeforces.com/group/MWSDmqGsZm/contest/219774/problem/C)
## Code
```
#include <bits/stdc++.h>
using namespace std;
int main() {
    int n;
    cin >> n;
    vector <long long> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    for (int i = 0; i < n; i++) {
        if (a[i] > 0) cout << 1;
        else if (a[i] < 0) cout << 2;
        else cout << 0;
        if (i != n - 1) cout << " ";
    }
}
```
