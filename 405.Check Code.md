# N. Check Code
- [Codeforces](https://codeforces.com/group/MWSDmqGsZm/contest/219774/problem/N)
## Code
```
#include <bits/stdc++.h>
using namespace std;
int main() {
    int a, b;
    cin >> a >> b;
    string s;
    cin >> s;
    bool f = true;
    if (s[a] != '-') f = false;
    else {
        for (int i = 0; i < s.size(); i++) {
            if (i == a) continue;
            if (s[i] >= '0' && s[i] <= '9') continue;
            else {
                f = false;
                break;
            }
        }
    }
    if (f) cout << "Yes";
    else cout << "No";
}
```
