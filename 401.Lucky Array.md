# J. Lucky Array
- [Codeforces](https://codeforces.com/group/MWSDmqGsZm/contest/219774/problem/J)
## Code
```
#include <bits/stdc++.h>
using namespace std;
int main() {
    int n;
    cin >> n;
    vector <int> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    int mini = *min_element(a.begin(), a.end());
    int cnt = 0;
    for (int i = 0; i < n; i++) {
        if (a[i] == mini) cnt++;
    }
    if (cnt & 1) cout << "Lucky";
    else cout << "Unlucky";
}
```
