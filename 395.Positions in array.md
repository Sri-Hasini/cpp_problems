# D. Positions in array
- [Codeforces](https://codeforces.com/group/MWSDmqGsZm/contest/219774/problem/D)
## Code
```
#include <bits/stdc++.h>
using namespace std;
int main() {
    int n;
    cin >> n;
    vector <int> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    for (int i = 0; i < n; i++) {
        if (a[i] <= 10) {
            cout << "A[" << i << "] = " << a[i] << endl;
        }
    }
}
```
