# F. Reversing
- [Codeforces](https://codeforces.com/group/MWSDmqGsZm/contest/219774/problem/F)
##
```
#include <bits/stdc++.h>
using namespace std;
int main() {
    int n;
    cin >> n;
    vector <int> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    for (int i = n - 1; i >= 0; i--) cout << a[i] << " ";
}
```
