# A. Summation
- [Codeforces](https://codeforces.com/group/MWSDmqGsZm/contest/219774/problem/A)
## Code
```
#include <bits/stdc++.h>
using namespace std;
int main() {
    int n;
    cin >> n;
    vector <long long> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    long long sum = accumulate(a.begin(), a.end(), 0LL);
    cout << abs(sum);
}
```
