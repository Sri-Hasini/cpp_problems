# O. Fibonacci
- [Codeforces](https://codeforces.com/group/MWSDmqGsZm/contest/219774/problem/O)
## Code
```
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n;
    cin >> n;
    long long a = 0, b = 1;
    long long c;
    if (n == 1) cout << 0;
    else if (n == 2) cout << 1;
    else {
        for (int i = 3; i <= n; i++) {
            c = a + b;
            a = b;
            b = c;
        }
        cout << c;
    }
}
```
