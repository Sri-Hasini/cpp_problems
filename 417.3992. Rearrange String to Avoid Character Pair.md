# 3992. Rearrange String to Avoid Character Pair
- [Leetcode](https://leetcode.com/problems/rearrange-string-to-avoid-character-pair/description/)
## Code 
```cpp
class Solution {
public:
    string rearrangeString(string s, char x, char y) {
        int cnt1 = 0, cnt2 = 0;
        for (int i = 0; i < s.size(); i++) {
            if (x == s[i]) cnt1++;
            else if (y == s[i]) cnt2++;
        }
        string t;
        for (int i = 0; i < cnt2; i++) t += y;
        for (int i = 0; i < s.size(); i++) {
            if (s[i] != x && s[i] != y) t += s[i];
        }
        for (int i = 0; i < cnt1; i++) t += x;
        return t;
    }
};
```
